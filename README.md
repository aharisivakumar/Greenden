# 🌿 Greenden - Flower Sale Website

Greenden is a responsive flower shop website built using **HTML** and **Tailwind CSS**. The project showcases a clean and modern user interface for browsing flower products and contacting the store.

## 🚀 Features

- 🏠 Home Page
  - Responsive Navigation Bar
  - Hero Section
  - Search Bar
  - Best Seller Section
  - Customer Reviews
  - Newsletter Subscription
  - Footer

- 🌸 Products Page
  - Collection of flower products
  - Product cards with images
  - Responsive layout

- 📞 Contact Page
  - Contact Form
  - User-friendly design

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS

## 🎯 Learning Outcomes

This project helped me practice:

- HTML page structure
- Tailwind CSS utility classes
- Responsive web design
- Navigation between multiple pages
- Image layout and styling
- Building a modern landing page
